# 🗓️ Planilla Semanal – Supermercado Átomo

Este proyecto es una aplicación web pensada para organizar de manera simple y visual los horarios semanales del personal dentro de un supermercado.

Nace como una solución práctica a un problema real: la dificultad de armar, visualizar y mantener actualizados los turnos de trabajo sin depender de planillas manuales o sistemas complejos.

---

## 🎯 Objetivo

Facilitar la gestión de horarios del personal, permitiendo:

- Registrar empleados por día
- Asignar horarios de entrada y salida
- Marcar francos de forma clara
- Organizar cajeros por turno
- Gestionar sectores específicos (fiambrería, carnicería, textil, etc.)
- Tener una vista resumida de toda la semana

Todo en una sola interfaz, rápida y fácil de usar.

---

## ⚙️ Funcionalidades principales

- 📅 Selección de semana automática
- 👥 Gestión dinámica de empleados (agregar / eliminar)
- ⏰ Carga de horarios por empleado
- ✅ Control de francos
- 🧾 Asignación de cajeros por turno (mañana, siesta y tarde)
- 🏬 Organización por sectores
- 📊 Resumen semanal con vista general
- 💾 Exportación de datos en formato JSON
- 🖨️ Opción de impresión optimizada

---

## 💡 Enfoque

La aplicación está diseñada con una filosofía clara:

> Ser rápida, clara y funcional en el día a día.

No busca ser un sistema complejo, sino una herramienta ágil que permita tomar decisiones rápidas y reducir errores en la organización del personal.

---

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3 (responsive y adaptable a mobile)
- JavaScript Vanilla (sin frameworks)

---

## 🚀 Uso

1. Seleccionar la semana
2. Completar los empleados por día
3. Asignar horarios y francos
4. Organizar cajeros y sectores
5. Revisar el resumen semanal
6. Exportar o imprimir la planilla
